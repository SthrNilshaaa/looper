## About
Looper Player is a high-performance, modern music player designed specifically for the Linux desktop. It combines a stunning Glassmorphism-based UI with a powerful audio engine to provide a seamless and immersive listening experience.

## Features
- **Modern UI/UX**: Material 3 design with Glassmorphism effects and smooth animations.
- **Dynamic Theming**: Album-based dynamic colors that adapt to your music.
- **High Performance**: libmpv-powered playback engine supporting MP3, FLAC, AAC, WAV, OGG, and OPUS.
- **Smart Library**: Recursive folder scanning with automatic indexing for 50k+ tracks.
- **Fast Database**: Powered by Isar for near-instant search and filtering.
- **Rich Metadata**: Automatic extraction of album art and track information.
- **Desktop Ready**: Custom title bar integration and native Linux desktop experience.

## Installation

You can download the latest version of Looper Player from the [Releases](https://github.com/SthrNilshaaa/one_player/releases) page. We provide several formats to ensure compatibility with all Linux distributions:

### 📦 Debian / Ubuntu (`.deb`)
Recommended for Debian, Ubuntu, Linux Mint, Pop!_OS, etc.
```bash
sudo apt install ./looper-player.deb
```

### 📦 Fedora / RHEL / openSUSE (`.rpm`)
Recommended for Fedora, openSUSE, and other RPM-based distros.
```bash
# Fedora
sudo dnf install ./looper-player.rpm
# openSUSE
sudo zypper install ./looper-player.rpm
```

### 🚀 AppImage (Universal)
The most universal format. Works on almost any distribution without installation.
1. Download the `.AppImage` file.
2. Make it executable: `chmod +x looper-player.AppImage`
3. Run it: `./looper-player.AppImage`

### 📂 Portable Tarball (`.tar.gz`)
Great for "extract and run" usage or custom installations.
1. Extract the archive: `tar -xzvf looper-player-linux.tar.gz`
2. Enter the directory: `cd universal`
3. Run the app: `./run.sh`
